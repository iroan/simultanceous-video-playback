# 项目功能
1. 开发一个视频同步播放的Android软件，方便我(me)和女友(gf)看电影
2. 一个story是：
   1. 打开本软件，同步播放电影文件
   2. 打开微信，开源进行声音交流
   3. 能够听到电影的声音和微信中的声音
